Cropper

Teste do plugin jQuery para um Cropper

Demonstração

Clica no Enviar Imagem pra fazer o upload e aí abrir o Cropper

Tu ainda pode clicar no thumb, para editar de novo.

Thumb da imagem Original

Imagem final

Básico

            $("#cropper").cropper({
                imagemOriginal : imagem,
                imagemFinal    : 'Mudei a imagem final',
                quandoSalvar   : function(arquivoFinal) {alert(arquivoFinal)}
            });

Parâmetros

        var defaults = {
            largura         : 500, //largura da imagem final
            altura          : 500, /altura da imagem final
            imagemOriginal  : null, //imagem original a ser editada
            imagemFinal     : 'novaImagem.jpg', //nome do arquivo de imagem pra retorno
            filtros         : true, //habilita os filtros
            getWH           : 'getWH.php', //pra pegar largura e altura da imagem
            preVisualizar   : 'previsualizar.php',
            wImg            : 0, // se getWH for null, definir wImg = largura da imagem original
            hImg            : 0, // se getWH for null, definir hImg = altura da imagem original
            quandoSalvar    : function(arquivoFinal) {}, //função de retorno quando clicar em ~salvar~
            quandoCancelar  : function(arquivoFinal) {} //função de retorno quando clicar em ~usar original~
        };
